# AI_DDR_Gen
AI management to generate DDR reports through inspection data

## Overview
This project converts inspection and thermal data into a structured Detailed Diagnostic Report (DDR).

## Features
- Extracts observations from raw data
- Organizes area-wise issues
- Identifies root causes
- Generates client-friendly report

## Output
HTML report (ddr_report.html)

## Note
Due to API quota limitations, output was generated manually while keeping the AI workflow intact. API keys are not included for security reasons.
